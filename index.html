Exploring My Passion for Technology and Web Development

Throughout my life, technology has been a driving force, leading me to build and troubleshoot computers and craft innovative websites. My journey through the digital realm has been marked by continuous growth and exploration.

Staying current in the ever-evolving tech landscape is more than a hobby; it's a fundamental aspect of my lifestyle. I thrive on engaging with cutting-edge technologies, consistently pushing boundaries, and creating digital experiences that leave a lasting impact.

My recent professional journey took an exciting turn as I gained valuable experience at North River Integrations, a dynamic software engineering company. As a key member of the team, I contributed to the development of web-based applications, honing my coding and problem-solving skills. Each day was an opportunity for learning and growth.

Building on this experience, I had the privilege of constructing the company's website from the ground up, showcasing my passion for web development. This venture has been truly remarkable, and I eagerly anticipate the unfolding chapters of this enriching journey.

College Lab Examples

Lab 1: Researching Languages

In this lab, I embarked on a journey to delve into the world of HTML, HTML5, and CSS. The task was to create a web page for each topic, summarizing my newfound knowledge. Each article contained well-written paragraphs and made use of tags covered in Chapter 1. This assignment not only deepened my understanding of web technologies but also honed my skills in structuring and presenting information on the web.

Lab 2: Creating a Web Page about a Chosen Topic

In this lab, I chose a topic close to my heart and created a well-structured web page ("index.html"). It featured a header, three sections with tags like article, section, and aside, and a navigation section for easy access. I used h1-h6 and p tags to ensure a clear and informative layout. The assignment covered tags from chapters 1 to 3, allowing me to enhance my web page creation skills and share my passion on the chosen topic.

Lab 3: Altering lab 3 by adding another page

In this lab, I improved my Lab 2 web page with images and text formatting. I also created a new page focusing on a different aspect, linked the two pages together, and uploaded the files for evaluation. This task enhanced my skills in multimedia elements and interlinking web pages.

Lab 4: CSS Styling for Website Transformation

In this lab, I creatively crafted a CSS file to enhance the appearance of my Lab 3 web pages. The impactful CSS changes transformed the overall look and feel, showcasing my growing proficiency in styling web content. I uploaded the CSS file to Dropbox, highlighting the significance of CSS in elevating web page aesthetics.

Lab 5: Analyzing Website Design Techniques

In this lab, I researched web design principles and identified two well-designed websites and two poorly designed ones. I then created a web page with external CSS, summarizing the principles I learned and providing links to the example sites with explanations of their design quality. This practical exercise enhanced my understanding of web design.

Lab 6: Linking Different CSS to Identical HTML Files

In this lab, I Explored CSS's creative potential in web design by creating two pages with identical content but drastically different styles. Experimented with fonts, colors, and layouts, showcasing growing mastery of CSS styling techniques.

Lab 7: Extensive CSS Formatting

Lab 7 was all about pushing the boundaries set in Lab 6. I took the familiar HTML code and unleashed a wave of creativity, transforming two pages into distinct visual experiences. The challenge? Redefine the UI, keeping the HTML core intact.

Lab 8: Mobile Web & Accessibility

Lab 8 let me embark on a design journey, creating a page that embraces good design principles for mobile web development. Through media queries and responsive techniques, the page dynamically adapts its layout for both desktop and mobile. In the second part, I delved into Web Accessibility Guidelines, crafting an inclusive model that ensures universal access.

Lab 9: User Interation with Server Side Scripts

In this lab, I implemented a server-side script provided by my professor. The goal was to create a practical web page allowing users to input specific values. These inputs were then smoothly transmitted to an external website, transforming the webpage into an interactive platform with enhanced functionality.

Lab 10: Data Table Design

Lab 10 was a hands-on experience where I crafted a data table featuring statistics from various countries. The challenge? Organizing the data with precision and ensuring a visually appealing design. I meticulously structured the table for optimal readability and extended the effort to make it equally engaging on mobile. Additionally, I provided a transparent link to the data source, enhancing the transparency of the information presented.

Lab 11: Responsive Login/ Register Design using JavaScript

In Lab 11, I designed a sleek login/homepage screen featuring a responsive login menu using JavaScript. The screen not only looks neat but also adapts seamlessly to different devices. I added functionality, allowing users to access different pop-ups for actions like "Forgot Password" and registration, enhancing the overall user experience.
